# AI Resume Builder

A modern **AI-powered resume builder** that allows users to create, enhance, and download professional resumes effortlessly. Built with a **MERN stack (MongoDB, Express, React, Node.js)** and integrated with AI capabilities for smart suggestions and resume enhancement.

---

## Features

- **User Authentication:** Secure signup and login system.
- **Resume Creation:** Add personal information, education, experience, skills, and projects.
- **AI-Powered Enhancements:** Improve resume summaries, job descriptions, and skills using AI suggestions.
- **Upload & Edit Resumes:** Users can upload existing resumes and edit them within the app.
- **Responsive Design:** Works smoothly on desktops, tablets, and mobile devices.
- **Download Resume:** Export resumes in PDF format for professional use.

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, Lucide Icons
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI Integration:** OpenAI API (for enhancing resumes)
- **Version Control:** Git & GitHub
